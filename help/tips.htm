Tips and Keyboard Shortcuts

Keyboard Shortcuts

ShortcutAction
Ctrl+PPrint the current image

Mouse Actions

ActionResult
Scroll wheel upZoom in (+15 % per notch, anchored to cursor)
Scroll wheel downZoom out (−15 % per notch, anchored to cursor)
Left-click + dragPan the image
Click folder in treeOpen first image in folder
Click image file in treeOpen that image

Tips

  • To reset the zoom to fit-to-window, click the image name in the folder tree again.
  • The title bar always shows the filename of the current image.
  • Expand a drive or folder by clicking its arrow — contents are loaded on demand, so large drives do not slow down the application.
  • Image files (JPG, PNG, BMP, GIF, TIFF, WebP) are shown directly in the tree alongside folders, so you can browse images without leaving the tree.
  • Print and Print Preview are enabled only when an image is loaded.
