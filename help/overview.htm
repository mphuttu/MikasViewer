Quick Start

  1. Use the folder tree on the left to navigate your drives and folders.
  2. Click a folder to automatically display the first image in it.
  3. Click an image file in the tree to open it directly.
  4. Use the mouse wheel to zoom and drag to pan.
  5. Print with File → Print or preview with File → Print Preview.
Tip: The current image filename is always shown in the title bar.
