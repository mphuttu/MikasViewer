Browsing and Viewing Images

The Folder Tree

The left panel shows an Explorer-style folder tree. All logical drives on your computer are listed at the root level with their Windows display names (for example, Windows (C:)).

  • Click the arrow or + next to a drive or folder to expand it. Subfolders and image files are loaded on demand.
  • Image files are shown directly inside folders — you do not need to open a separate file dialog.

Opening an Image

ActionResult
Click a folder Displays the first image found in that folder.
Click an image file Displays that image immediately.

Image Display

When an image is opened it is automatically scaled to fit the window (up to 100 % of its original size). If the image is smaller than the window, it is shown at its actual size and centered.

The window title bar shows the filename of the currently displayed image.

Tip: Resize the application window — the image recenters itself automatically.
