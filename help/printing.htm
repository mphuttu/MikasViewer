Printing

Print

To print the currently displayed image, choose File → Print... or press Ctrl+P.

The standard Windows Print dialog opens, where you can select a printer, set the number of copies, and configure printer properties.

Print Preview

To preview the print output before sending it to the printer, choose File → Print Preview.

The preview window shows exactly how the image will appear on paper. Use the toolbar buttons to zoom in/out on the preview or to switch between one-page and two-page view. Click Print to print directly from the preview, or Close to return to the main window.

Page Layout

The image is printed as follows:

  • Scaled to fill the printable area as large as possible while preserving the aspect ratio.
  • Centered on the page.
  • A small margin (approx. 5 % on each side) is added for aesthetic appearance.

High-quality bicubic interpolation is used to ensure sharp output at any print resolution.

Tip: Print and Print Preview are only available when an image is currently displayed. They are greyed out when no image has been loaded.

Print Setup

To change the paper size, orientation, or default printer settings, choose File → Print Setup....
