Supported Image Formats

MikasViewer can open the following image formats:

Extension(s)FormatNotes
.jpg  .jpegJPEG Loaded via GDI+. Most common photo format.
.pngPNG Loaded via GDI+. Supports transparency (alpha channel).
.bmpBitmap Loaded via GDI+. Standard Windows bitmap.
.gifGIF Loaded via GDI+. Only the first frame is displayed.
.tif  .tiffTIFF Loaded via GDI+. Supports multi-page files (first page shown).
.webpWebP Loaded via Windows Imaging Component (WIC). See note below.
WebP on Windows 10: WebP support requires the WebP Image Extensions app from the Microsoft Store. On Windows 11 the codec is built in — no additional installation is needed.

How Formats Are Detected

MikasViewer detects image files by their file extension (case-insensitive). Files with the extensions listed above are shown in the folder tree and can be opened by clicking.
